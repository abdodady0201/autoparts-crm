# 🔧 قطعتي - نظام إدارة محل قطع غيار السيارات

تطبيق ويب بسيط لإدارة مبيعات ومخزون محل قطع غيار السيارات. يعمل مباشرة في المتصفح بدون خادم أو قاعدة بيانات خارجية.

## ✨ المميزات

- 🛒 **بيع سريع** — إضافة قطع للسلة بضغطة واحدة، تأكيد البيع في ثوانٍ
- 📦 **إدارة المخزون** — تتبع الكميات مع تنبيه عند النفاد
- 🧾 **سجل المبيعات** — عرض وطباعة وصل لكل عملية بيع
- 🚛 **الموردون** — حفظ بيانات الموردين
- 📊 **لوحة التحكم** — إحصائيات اليوم والشهر
- 💾 **حفظ تلقائي** — البيانات محفوظة في المتصفح (localStorage)

## 🗂️ هيكل الملفات

```
autoparts-crm/
├── index.html          # الصفحة الرئيسية
├── css/
│   └── style.css       # جميع أنماط التصميم
├── js/
│   ├── db.js           # قاعدة البيانات (localStorage)
│   ├── utils.js        # دوال مساعدة
│   ├── app.js          # التنقل بين الصفحات
│   ├── dashboard.js    # لوحة التحكم
│   ├── sell.js         # البيع والسلة
│   ├── inventory.js    # المخزون
│   ├── orders.js       # سجل المبيعات
│   └── suppliers.js    # الموردون
└── README.md
```

## 🚀 النشر المجاني على GitHub Pages

1. ارفع المجلد على GitHub
2. اذهب إلى **Settings → Pages**
3. اختر **Branch: main** ثم اضغط **Save**
4. رابطك سيكون: `https://اسمك.github.io/autoparts-crm`
